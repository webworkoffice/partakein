上級者向けの解説

このエラーは、次のような場合に起こる可能性があります。

  • 多重投稿防止のためのトークンチェックにひっかかった。
  • 必要なパラメータがサーバーに渡されていない。(POST リクエストを新しいタブで開きながら投げようとしたときに起きる場合があります。)
  • 登録されていない event ID などが渡された。
  • CSRF (Cross-site Request Forgery) の攻撃にひっかかった。
